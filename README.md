# Quick Label Generator — How to Use

## Opening the File

- **Option 1** — Download from SharePoint, then double-click to open.
- **Option 2** — On SharePoint, click **Open** → **Open in app**.

---

## Filling Out the Form

All info comes from the chemical's SDS. Work top to bottom.

| Field | What to enter |
|---|---|
| Product Identifier | Full product name from the SDS |
| Signal Word | DANGER, WARNING, or None (SDS Section 2) |
| Hazard Statements | H-codes from SDS Section 2 |
| Precautionary Statements | P-codes from SDS Section 7 |
| First Aid | By exposure route from SDS Section 4 |
| GHS Pictograms | Check every pictogram listed in SDS Section 2 |
| NFPA 704 | Ratings 0–4 for Health, Flammability, Instability, and Special (SDS Section 9) |
| HMIS | Ratings 0–4 for Health, Flammability, Reactivity, and a PPE letter code A–K (SDS Section 8) |
| Supplier Info | Company, address, emergency phone, your name, date |

---

## Label Size

Use the toggle at the top to switch before printing:

| Mode | Paper | Best for |
|---|---|---|
| Full (10×7.5 in) | Letter landscape | Large or secondary containers |
| Sticker (3×4 in) | Small label stock | Small containers or pipes |

---

## Printing

1. Check the preview on the right looks correct.
2. Click **Print / Save as PDF**.
3. Select your printer and print.
